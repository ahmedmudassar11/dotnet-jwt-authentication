# 🔐 .NET JWT Authentication

A clean and secure **ASP.NET Core Web API** implementing **JWT Authentication** with **Access Tokens** and **Refresh Tokens**. This project demonstrates a production-style authentication flow with secure user authentication, authorization, and protected API endpoints.

## ✨ Features

- JWT Authentication
- Access & Refresh Tokens
- User Registration & Login
- Role-Based Authorization
- Claims-Based Authentication
- Secure Password Hashing
- Entity Framework Core
- SQL Server
- Swagger API Documentation

## 🛠️ Tech Stack

- ASP.NET Core Web API
- C#
- Entity Framework Core
- SQL Server
- JWT Bearer Authentication
